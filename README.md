
### Daum 영화 순위 썸네일 다운로더
Selenium을 사용하여 Daum 검색 결과 페이지에서 특정 연도의 영화 순위 썸네일을 다운로드하는 코드입니다. 
사용자가 지정한 연도의 영화 순위를 검색하고, 해당 순위의 영화 썸네일 이미지를 다운로드합니다.
